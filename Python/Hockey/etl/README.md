
# word_cloud

Examples of word clouds from various Hockey Hall of Fame Speeches.

## Features

- Light/dark backgrounds
- Use of color palettes
- FontAwesome shapes

## Installation 

Install my-project with npm

```bash 
  npm install my-project
  cd my-project
```
    
## Tech Stack

Python 3.9x, Stylecloud, FontAwesome 5.0

Clone the project

```bash

```

```bash
  cd my-project
```

